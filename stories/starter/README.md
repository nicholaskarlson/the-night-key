# Your Story

This folder was generated by `btg init-story`.

## Run

From the repo root (or anywhere that has `btg` installed):

```bash
btg play --scenes ./scenes.yaml
```

## Validate (recommended)

```bash
btg lint --strict --scenes ./scenes.yaml
```

## Edit

Open `scenes.yaml` in a text editor and start writing.

Tips:
- Keep scene ids in `lower_snake_case`.
- Use `terminal: true` on endings.
- Use `flags` + `requires_flags` to unlock memory threads.
