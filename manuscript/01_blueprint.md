# The Night Key — Book Blueprint (Phase 2)

This file is intentionally a stub scaffold so we can fill it in step-by-step.

## Working title options
- The Night Key
- The Night Key: A Build-Along Interactive Fiction Book

## Subtitle options
- Write Interactive Fiction in YAML with `btg` — by building one real story
- A non-programmer’s build-along guide to interactive fiction (proof-first)

## Audience
- Non-programmers who can edit files and run commands
- Beginner writers who want a concrete template

## Promise
By the end you will:
- understand how the engine works,
- have built a substantial story, and
- be able to pack and share it deterministically.

## Outline (placeholder)
1. Install + play the seed
2. First branch + first flag gate
3. State deltas (energy/warmth/support/guilt)
4. Designing scenes as cards
5. Replay scripts + transcript comparison
6. More endings + polish
7. Pack + verify + share on GitHub
